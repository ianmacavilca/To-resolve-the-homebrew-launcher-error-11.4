Luego cree una carpeta llamada hblauncherloader
dentro ponga el archivo PaylUSA.bin
Luego inicie el 3ds y le volvera  a salir el error.
Despues,saque la SD y entre a la carpeta hblauncherloader,copie el archivo OLD-11-4-0-37-USA dentro de la carpeta;
tiene que copiar y reemplazar, por ultimo entre a la 3ds y cargara normalmente.

Si es de otra region puede entrar a la pagina
 http://smealum.github.io/3ds/
descargar OLD-11-2- (tu region) y cambiarle el nombre por OLD-11-4-0-37-(Tu region), luego realizar el procedimiento anterior.
